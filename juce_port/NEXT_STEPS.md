## 🔧 Полезные команды

### Сборка проекта
```powershell
# В Visual Studio:
# Build → Build Solution (Ctrl+Shift+B)
```

### Поиск собранного плагина
```powershell
Get-ChildItem "juce_port\juce_project\Builds" -Recurse -Filter "*.dll"
```

### Регистрация плагина в DAW
- Скопировать DLL в папку плагинов DAW
- Перезапустить DAW
- Найти "Creative MIDI Generator"

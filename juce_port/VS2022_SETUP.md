# Настройка Visual Studio 2022 для JUCE проекта

## 🎯 Цель: Скомпилировать и протестировать Creative MIDI Generator

## 📋 Предварительные требования

### 1. Установить Visual Studio 2022
**Необходимые компоненты:**
- ✅ Рабочая нагрузка "Разработка классических приложений на C++"
- ✅ MSVC v143 - VS 2022 C++ x64/x86 (последняя версия)
- ✅ Windows 10 SDK (последняя версия)

**Системные требования:**
- RAM: минимум 8GB
- Disk: минимум 10GB свободного места
- OS: Windows 10/11 64-bit

## 🚀 Пошаговая инструкция

### Шаг 1: Открыть проект в Visual Studio
```
Путь: juce_port\juce_project\Builds\VisualStudio2022\Creative MIDI Generator.sln
```

**Действия:**
1. Найти файл `.sln` в проводнике
2. Дважды кликнуть для открытия в VS2022
3. Подождать загрузки проекта

### Шаг 2: Настроить конфигурацию
**В верхней панели Visual Studio:**
- **Платформа:** `x64`
- **Конфигурация:** `Release` (для продакшена) или `Debug` (для отладки)

### Шаг 3: Исправить проблемы компиляции
**Если возникают ошибки памяти:**
1. **Project → Properties**
2. **C/C++ → Command Line**
3. **Additional Options:** добавить `/Zm500`
4. **Linker → Command Line**
5. **Additional Options:** добавить `/LARGEADDRESSAWARE`

### Шаг 4: Собрать проект
**В Visual Studio:**
1. **Build → Build Solution** (или `Ctrl+Shift+B`)
2. **Ждать завершения компиляции**
3. **Проверить Output окно на ошибки**

### Шаг 5: Найти скомпилированные файлы
**После успешной сборки:**
```
📁 juce_port\juce_project\Builds\VisualStudio2022\x64\Release\
├── ✅ CreativeMIDIGenerator.dll          # VST3 плагин
├── ✅ CreativeMIDIGenerator.exe          # Standalone приложение
├── ✅ CreativeMIDIGenerator.lib          # Библиотека
└── ✅ CreativeMIDIGenerator.exp          # Export файл
```

### Шаг 6: Тестирование

#### Тест 1: Standalone приложение
```powershell
# Запустить standalone версию
./CreativeMIDIGenerator.exe
```
**Ожидаемый результат:**
- Откроется окно приложения
- Можно будет генерировать MIDI паттерны

#### Тест 2: VST3 плагин
1. **Скопировать** `CreativeMIDIGenerator.dll` в папку плагинов DAW
2. **Перезапустить DAW**
3. **Найти плагин** "Creative MIDI Generator"
4. **Добавить** как MIDI эффект
5. **Протестировать** генерацию в DAW

## 🐛 Возможные проблемы и решения

### Проблема 1: "Cannot open file 'v142/143.build'"
**Решение:**
- Установить недостающие компоненты VS2022
- Обновить Windows SDK

### Проблема 2: Ошибки компиляции JUCE
**Решение:**
- Проверить пути к JUCE: `D:\PROG\JUCE`
- Перегенерировать проект в Projucer

### Проблема 3: Проблемы с памятью
**Решение:**
- Добавить флаги `/Zm500` и `/LARGEADDRESSAWARE`
- Увеличить виртуальную память Windows
- Закрыть другие приложения

### Проблема 4: Плагин не виден в DAW
**Решение:**
- Проверить правильность папки для плагинов
- Перезапустить DAW полностью
- Проверить, что DAW поддерживает VST3

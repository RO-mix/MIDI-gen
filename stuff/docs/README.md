# Creative MIDI Looper & Pattern Studio

## Обзор

Creative MIDI Looper & Pattern Studio - это эволюция Creative MIDI Generator в полноценную систему для создания, развития и управления музыкальными паттернами в реальном времени.

## Быстрый старт

### Текущая версия (Pattern Generator)
```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python main.py
```

### Будущая версия (Looper Studio)
```bash
# Установка расширенной версии
pip install -r requirements-dev.txt

# Запуск в режиме разработки
python main.py --dev-mode
```

## Основные возможности

### Сейчас доступно:
- ✅ Генерация паттернов (Random, Euclidean, Dual Euclidean)
- ✅ Поддержка музыкальных ладов
- ✅ Real-time MIDI вывод
- ✅ Система пресетов
- ✅ Визуализация паттернов

### В разработке:
- 🔄 MIDI Looper Engine
- 🔄 Pattern Variation Engine
- 🔄 Smart Loop Extension
- 🔄 AI-powered генерация

# Техническое задание: Dual Euclidean Generator v2.0

## Основные улучшения:

### 1. Гармонические отношения
- **Интервал между машинами**:
  - Параметр `interval: int` (0-12 полутонов)
  - Автоматический расчет ноты B относительно ноты A
  - Поддержка музыкальных интервалов (терции, кварты и т.д.)

### 2. Управление паттернами
- **Фазовый сдвиг**:
  - Параметр `phase_shift: float` (0-1)
  - Возможность сдвигать паттерн машины B относительно A
- **Синхронизация**:
  - Кнопка синхронизации шага машин
  - Режим "Master-Slave"

### 3. Динамика и акценты
- **Акценты**:
  - Параметр `accent_strength: float` (0-1)
  - Периодические акценты в паттерне
- **Динамика**:
  - Параметр `dynamic_range: float` (0-1)
  - Вариации velocity для создания более живого звучания

### 4. Полиритмические пресеты
- Набор предустановленных полиритмов:
  - 3:2, 4:3, 5:4 и т.д.
  - Возможность сохранять пользовательские комбинации

## Пример новых параметров:
```python
interval: int = 4  # Терция
phase_shift: float = 0.25  # Сдвиг на 25%
accent_strength: float = 0.3
dynamic_range: float = 0.5
```

## Изменения в классе:
```python
class DualEuclideanGenerator(BaseGenerator):
    def __init__(self):
        # Новые параметры
        self.interval = 4
        self.phase_shift = 0.0
        self.accent_strength = 0.0
        self.dynamic_range = 0.0
        self.sync_mode = 'Independent'  # 'Independent' или 'MasterSlave'
```

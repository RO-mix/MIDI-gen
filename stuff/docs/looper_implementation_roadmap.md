### API для генераторов
```python
class GeneratorWithLooperSupport:
    def __init__(self):
        self.looper_enabled = False
        self.looper_buffer_size = 4  # bars
        self.looper_effects = []
    
    def enable_looper(self, buffer_size=4):
        """Включить лупизацию для этого генератора"""
        self.looper_enabled = True
        self.looper_buffer_size = buffer_size
    
    def process_looper(self, midi_events):
        """Обработать MIDI через лупер"""
        if self.looper_enabled:
            return self.generation_looper.process(midi_events)
        return midi_events
```

### Параметры по умолчанию
- **MIDI Looper**: 4 бара, 1/16 квантизация, все каналы
- **Generation Looper**: 2 бара, 1x скорость, 0 полутонов pitch
- **Buffer size**: 1024 samples
- **Sync mode**: Internal clock

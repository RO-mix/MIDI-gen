# План интеграции MIDI лупера в существующий проект

## Текущее состояние GUI
- **main_window.py** содержит вкладочный интерфейс с вкладками "Генераторы", "Микшер", "Настройки"
- **Генераторы** - основная рабочая область с тремя генераторами (Random, Euclidean, Dual Euclidean)
- Каждый генератор имеет свои элементы управления и визуализацию

## Варианты интеграции лупера

### Вариант 1: Новая вкладка "Лупер"
**Плюсы:**
- Чистое разделение функциональности
- Простота реализации
- Место для расширенных функций лупера

**Минусы:**
- Переключение между вкладками может быть неудобным
- Теряется контекст при работе с генераторами

### Вариант 2: Интеграция в вкладку "Генераторы"
**Плюсы:**
- Все в одном месте
- Прямая связь между лупером и генераторами
- Быстрый доступ к вариациям

**Минусы:**
- Может перегрузить интерфейс
- Ограниченное пространство

## Рекомендуемый подход: Гибридная интеграция

### 1. Основной интерфейс лупера - новая вкладка
```
[Лупер] [Генераторы] [Микшер] [Настройки]
```

### 2. Быстрый доступ из вкладки "Генераторы"
- Кнопка "Записать в лупер" рядом с каждым генератором
- Мини-панель с последними паттернами

### 3. Элементы управления в вкладке "Лупер"

#### Панель записи
- **Кнопка записи** (красная круглая)
- **Индикатор времени** (оставшееся время записи)
- **Выбор длины** (1, 2, 4, 8 баров)
- **Квантование** (1/4, 1/8, 1/16, 1/32)

#### Список паттернов
- **Таблица паттернов** с колонками:
  - Название
  - Длина (в барах)
  - BPM
  - Дата создания
  - Источник (какой генератор записал)
- **Кнопки действий** для каждого паттерна:
  - Play/Pause
  - Вариации
  - Редактировать
  - Удалить

#### Панель вариаций
- **Список вариаций** для выбранного паттерна
- **Слайдер интенсивности** вариации (0-100%)
- **Предпросмотр** вариации

## Техническая реализация

### Новые классы GUI
1. **LooperPanel** - основная панель лупера
2. **PatternListWidget** - список паттернов
3. **VariationPanel** - панель вариаций
4. **QuickLooperButton** - кнопка быстрой записи в генераторах

### Интеграция с существующим кодом
```python
# В main_window.py добавить:
class MainWindow:
    def __init__(self):
        # ... существующий код ...
        self.create_looper_tab()
        
    def create_looper_tab(self):
        # Создание вкладки лупера
        self.looper_panel = LooperPanel(self.sequencer)
        self.tabs.addTab(self.looper_panel, "Лупер")
```

### Минимальные изменения в существующие файлы
- **main_window.py**: добавить вкладку и кнопки быстрой записи
- **generator_panel.py**: добавить кнопки записи для каждого генератора
- **sequencer.py**: добавить методы для работы с лупером

# Техническая архитектура Creative MIDI Generator

## Обзор системы

Creative MIDI Generator - это многопоточное приложение для генерации MIDI-паттернов с поддержкой различных алгоритмов и музыкальной теории. Приложение построено на модульной архитектуре с четким разделением ответственности между компонентами.

## Архитектурные слои

### 1. Генераторы паттернов (`stuff/src/generators/`)
- **BaseGenerator** - абстрактный базовый класс для всех генераторов
- **RandomGenerator** - случайная генерация нот с учетом музыкальной теории
- **EuclideanGenerator** - евклидовые ритмы
- **DualEuclideanGenerator** - комбинированные евклидовые ритмы

### 2. Движок секвенсора (`stuff/src/engine/`)
- **Sequencer** - высокоточный секвенсор (1мс точность)
- **MIDIHandler** - обработка MIDI-сообщений
- **PatternManager** - управление паттернами и пресетами

### 3. Музыкальная теория (`stuff/src/theory/`)
- **Scales** - 14 музыкальных ладов
- **Durations** - система длительностей
- **NoteFilter** - фильтрация нот по ладам

### 4. Пользовательский интерфейс (`stuff/src/gui/`)
- **MainWindow** - главное окно на Dear PyGui
- **ControlPanel** - панель управления параметрами
- **PatternView** - визуализация паттернов

### 5. Конфигурация и пресеты
- **config.json** - основная конфигурация
- **presets/** - сохраненные пресеты паттернов

## Потоковая модель

Приложение использует три основных потока:
1. **GUI Thread** - обработка пользовательского интерфейса
2. **Sequencer Thread** - точное воспроизведение паттернов
3. **MIDI Thread** - обработка MIDI-сообщений

## Точки расширения

- Новые генераторы паттернов через наследование `BaseGenerator`
- Новые лады в `Scales`
- Пользовательские фильтры нот
- Расширенные MIDI-эффекты
